Some things to try

  1. Start the dev server: webstir watch
  2. Edit this page: src/frontend/pages/home/index.html
  3. Add a doc: src/frontend/content/my-doc.md
  4. Enable smoother navigation: webstir enable client-nav
  5. Publish output: webstir publish
